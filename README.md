## 실행 방법

### Prerequisites

```
npm install npm@latest -g
```

### Installation

Clone the repo

```
git clone https://github.com/woowa-techcamp-2022/web-fleamarket-9.git
```

Install NPM packages

```
cd frontend
npm install


cd server
npm install
```

### Environment variable

frontend

```
/frontend/.env.sample
```

backend

```
/backend/.env.sample
```
